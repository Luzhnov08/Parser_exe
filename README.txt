Консольное десктоп-приложение .net core для анализа текста на веб-странице.

1)Распаковать содержимое архива в папку на локальном ПК для нужной ОС.
2)Запустить ParserTask.exe
3)Ввести URL-адрес интересующей страницы
4)В результате в консоли будет отражена детальная статистика и итог в последней строке.
5)Нажать любую клавишу для выхода из приложения

Приложение парсит веб-страницу с помощью библиотеки.
В случае ошибки приложения, детализация будет описана в файле Logging.txt в каталоге размещения приложения.
Приложение также записывает статистику в локальную БД sqlite.
